# PRUEBA QPH

**Autor:** Robinson Ortega

## Descripción

Proyecto de gestión de productos, usuarios y ventas.

## Estructura

- [backend](./backend-qph): API Rest hecho con Java Spring Boot
- [frontend](./frontend-qph): Aplicación web construida con React

## Tecnologías

- [Java](https://www.java.com/es/)
- [Spring Boot](https://spring.io/projects/spring-boot)
- [Angular](https://angular.io/)
- [Docker](https://www.docker.com/)
- [PostgreSQL](https://www.postgresql.org/)

## Instrucciones

1. Clonar el repositorio.
2. Ejecutar el comando `docker compose up -d` para levantar los contenedores.
3. Abrir el navegador y acceder a la aplicación en la dirección `http://localhost:80`.
